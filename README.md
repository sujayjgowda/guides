# Interview Prep Hub — PV & SQL Mastery

A comprehensive, offline-accessible study guide hub designed for Pharmacovigilance (PV) professionals and Data Analysts. This dashboard serves as a central portal to access deep-dive interview preparation materials, regulatory frameworks, SQL mastery guides, and real-world scenarios.

## 🌟 Features

- **Self-Contained & Offline-Accessible:** All guides are designed to be run locally without requiring an internet connection.
- **Modern UI/UX:** Built with a beautiful, responsive dark-mode interface featuring animated gradient backgrounds, subtle grid lines, floating particles, and interactive hover effects.
- **Dynamic Clock Widget:** A custom-designed analog and digital clock widget positioned in the top-right corner.
- **Quick Navigation:** Easy access to comprehensive study guides via visually distinct cards with detailed tags and descriptions.

## 📚 Available Guides

### 1. Pharmacovigilance Interview Mastery
A complete reference for PV professionals, ready for IQVIA / CRO interviews.
- **Scope:** 14 Sections · Deep Dive
- **Topics Covered:** PV fundamentals, E2B R2/R3, ICSR processing, GAMP 5, signal detection, Oracle Argus workflows, regulatory frameworks, and mock interview scenarios.
- **Key Tags:** `ICH E2B`, `EudraVigilance`, `GAMP 5`, `ICSR`, `Oracle Argus`

### 2. SQL End-to-End Guide
A complete roadmap from zero to hero for data analysis in the PV domain.
- **Scope:** 15 Sections · Zero to Hero
- **Topics Covered:** Every clause, function & pattern — from SELECT to window functions. Includes JOINs, subqueries, CTEs, aggregate functions, indexing, DDL/DML, and real interview Q&A with PV-domain examples.
- **Key Tags:** `SELECT · JOINs`, `Subqueries · CTEs`, `Window Functions`, `Performance`

### 3. GAMP 5 Validation Guide
A thorough guide to Good Automated Manufacturing Practice (GAMP 5) for the pharmaceutical industry.
- **Scope:** Comprehensive · Validation
- **Topics Covered:** Risk-based approaches, computer system validation life cycle, and compliance in the life sciences.
- **Key Tags:** `Compliance`, `Validation`, `Risk Assessment`

## 🛠️ Technology Stack

- **HTML5:** Semantic structure and content.
- **Vanilla CSS3:** Custom styling, CSS variables for scalable theming, flexbox/grid layouts, and keyframe animations (built entirely without external CSS frameworks).
- **Vanilla JavaScript:** Dynamic functionality including the interactive clock widget and floating background particle generation.
- **Typography:** DM Serif Display, DM Mono, Syne (via Google Fonts).

## 🚀 Getting Started

1. Clone or download the repository to your local machine.
2. Open `index-2.html` (or your main index file) in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Click on the respective guide cards to navigate to the individual study materials.
